TextModule.print (surface, font, text, x, y, w, h)
Print a text on a surface. Font should be a string specified in registeredFonts.

Parameters

  • surface: The surface on which we want to print the text.
  • font: The font we want to use when printing
  • text: The text to be printed
  • x: Start position for printing on the surface x-axis
  • y: Start position for printing on the surface y-axis
  • w: The width to use for printing the text
  • h: The height to use for printing the text

Return values:

  1. i Position of specific character
  2. x X-coordinate
  3. last_i Position of last character on text
