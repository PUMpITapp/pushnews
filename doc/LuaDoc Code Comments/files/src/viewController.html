  • view: The view module which should be added to the identifier

viewController:onKey (key, state)
This allows to have one onKey function per view. Each view has it's own onKey function and the right onKey function is exectued depending on the current view that is shown on the screen.

Parameters

  • key: keyboard input
  • state: state of the key, down = pressed, up = released
viewController:presentView (identifier)
Present a view to the screen. Each view should have a viewDidLoad and drawView method. viewDidLoad is called when the view is loaded This should allow to redraw the view as it was before presenting another view.

Parameters

  • identifier: Name of the view
